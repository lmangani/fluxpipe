## `json` package

Package json provides tools for working with JSON.

Import the `json` package:

```flux
import "json"
```

### Functions

### `encode()`

encode converts a value into JSON bytes.

#### Function type signature

```flux
(v: A) => bytes
```

#### Parameters

| Parameter | Description | Required |
| --- | --- | --- |
| v | v: Value to convert. | Yes |
